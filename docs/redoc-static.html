Create password

Allows a new user to create their password in order to grant them access.

Request Body schema: application/json
id
required
string <objectId>

Unique vendor ID

token
required
string <uuid>

Vendor token

password
required
string <password>

New password for the vendor

confirmPassword
required
string <password>

Same password for confirmation

Responses

Production server

https://api.inlet.shop/vendor/password

Development server

http://127.0.0.1:8001/vendor/password

Authorizations:
VendorAuth
Request Body schema: application/json

tags
required
Array of strings

archived
required
boolean

Whether this item is marked as 'deleted' for the vendor. Generally should always be true on product creation.

required
Array of objects

List of the prices/variations of this product.
